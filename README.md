## Description

WeatherApp is a simple mobile application that allows users to check current weather conditions for selected cities. The application retrieves live data from an external API and presents it in a clear and user-friendly interface.

The project was created to practice working with REST APIs, asynchronous data handling, and Android application development.

---

## Features

* Search weather by city name
* Display current temperature
* Show weather conditions (e.g. clouds, rain, clear sky)
* Additional data such as humidity, pressure, and wind speed
* Basic error handling for invalid input or network issues
